As tags <Sub> e <Sup>

A sub define texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O.

A sup define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, como WWW1.


A tag <font>

Não suportado em HTML5.

A tag font foi usada no HTML 4 para especificar o tipo de fonte, o tamanho da fonte e a cor do texto.


Falando da tag <del>

A del define o texto que foi excluído de um documento. Os navegadores geralmente riscam o texto excluído.

Sobre a tag <p>

A tag <p> define um parágrafo.

Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada <p> elemento.
